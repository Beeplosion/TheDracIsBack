About

This project aims to analyze and represent Bram Stokers Dracula. It focuses on different historical locations, different historical locations, important characters, and mention of tecnology in the book. During the course of the project we had analysed the text, broke our anaysisation down by chapters, and desplayed the locations, characters, and tecnollogy we found in the book. If we were to continue with the project. We wuld look for cultural implications, references to medical practices or history, and how the location describe in the book related to actual locations and places during the time period the book takes place.

About: We set up rules that confined the text options inside elements such as date and char. We did this to make sure the information inside those key elements stayed consistent with the original text, as it would trigger an error if it were to be changed.

XSLT Link

About: The goal with the XSLT was to be able to show all the different technologies, locations and characters within a specific chapter. We did this by first creating an xml sheet that was marked up with all the different fields we were looking for. We then used XSLT to create an interactive analysis page that included a table that shows first each chapter, and then the corresponding attributes of interest from within that chapter. Each chapter than links to its corresponding chapter on the text page. The text page consists of the entire text of the book Dracula. Each mention of a technology, character or location is highlighted and underlined for better visibility. This was done like this to better help someone understand the world and culture that Dracula took place in.
